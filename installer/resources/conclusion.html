Getting Started

  1. Start the VPN client service:
    sudo gnosis_vpn -c /etc/gnosisvpn/config.toml
    This starts the VPN daemon. Keep this terminal open while using the VPN.
  2. Check the connection status:
    gnosis_vpn-ctl status
  3. Connect to a VPN location:
    gnosis_vpn-ctl connect 0x7220CfE91F369bfE79F883c2891e97407D7a4D48
    This example connects to the USA (Iowa) location on Rotsee network.

HTTP(S) Proxy Configuration

After establishing a VPN connection, configure your applications to use:

HTTP(S) Proxy: 10.128.0.1:3128

This allows you to browse anonymously through the VPN connection.

Configuration File

Your VPN settings are stored at: /etc/gnosisvpn/config.toml

You can edit this file to customize destinations, add intermediates, or configure advanced options.

Command-Line Tools

The command-line tools have been installed to /usr/local/bin and are available in your PATH:

  • gnosis_vpn - Main VPN client daemon
  • gnosis_vpn-ctl - Control utility for managing connections

You can run these commands from any terminal window.
