Installation Details

This installer will:

  • Download the latest Gnosis VPN binaries from GitHub releases
  • Install command-line tools to /usr/local/bin/
  • Install GnosisVPN.app to /Applications/
  • Remove macOS quarantine attributes for proper execution
  • Generate a default configuration file with VPN destinations
  • Verify and install WireGuard tools if necessary

Installation Locations

Files will be installed to:

  • /usr/local/bin/gnosis_vpn - Main VPN client daemon
  • /usr/local/bin/gnosis_vpn-ctl - Control utility
  • /Applications/GnosisVPN.app - Graphical application
  • /etc/gnosisvpn/config.toml - Configuration file

Note: The installer requires administrator privileges to:

  • Install files to /Applications
  • Remove quarantine attributes from downloaded binaries
  • Install WireGuard tools if not present

Privacy: This installer downloads binaries from GitHub and does not collect or transmit any personal information.
