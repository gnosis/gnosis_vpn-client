Gnosis VPN Client Installer

Welcome! This installer will set up the Gnosis VPN client on your Mac, providing you with decentralized VPN connectivity through the HOPR network.

What will be installed:

✓ Gnosis VPN client daemon
✓ Gnosis VPN control utility (gnosis_vpn-ctl)
✓ Default configuration with multiple VPN locations

System Requirements

  • macOS 11.0 (Big Sur) or later
  • WireGuard tools (required - install via Homebrew if not present)
  • Approximately 50MB of available disk space
Estimated installation time: 2-3 minutes
